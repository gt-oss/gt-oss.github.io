Georgia Tech OSS

The home of open computing at Georgia Tech

About

GT OSS is a github organization for projects developed by students, staff, and faculty at the Georgia Institute of Technology. Any students, staff, or faculty may create repositories on github.com/gt-oss — email oss@gatech.edu using your @gatech.edu account and provide your github username to be added to the organization.

There are only a few guidelines we maintain within this organization:

  • Projects must be original, open source work.
  • Projects must contain a README file.
  • This work should be something you are proud to be associated with you and Georgia Tech. If you're not sure, it is best to host it on your own github account.

That's it! We encourage the use of the MIT license if those terms fit your needs, but this is not required.

Projects hosted on github.com/gt-oss are not endorsed by Georgia Tech in any way.

Happy coding!
